# 创作者资源重配分析

本分析聚焦资源配置效率：在流量和现金激励受限的前提下，平台是否把资源持续配置给了更可能形成稳定供给、留存和收入的创作者。

## 当前判断

- 主要损耗发生在首发后的持续经营阶段。
- Top 20% 创作者拿走 42.9% 曝光，只贡献 30.2% 收入，资源分配仍带有较强历史曝光惯性。
- 固定 Top 4,200 的历史回测下，新规则相对旧规则带来：
  - 30 日留存率：48.7% → 58.1%（+9.4ppt）
  - 单位激励收入：69.6 → 78.4（+12.7%）
  - 人均现金激励：550 → 625（+13.6%）
- 新旧名单重合率约 72.2%，调整幅度可控。

## 结论

离线诊断与同规模回测支持进入 creator 级分层灰度实验；是否扩量，由线上主指标与护栏决定。

## 页面结构

- `index.html`：首页，适合简历外链第一落点
- `report.html`：完整报告，给出证据链、口径取舍、回测解读和实验门槛
- `notebook.html`：分析过程页，展示问题—口径—代码—判断的结构
- `sql.html`：SQL 方案页，覆盖漏斗、分层、固定 Top N 回测、名单迁移和实验监控
- `reports/creator_value_analysis_report.md`：Markdown 报告
- `analysis.py`：示例脚本
- `outputs/*.csv`：决策分卡、候选口径取舍、实验设计卡、有效性威胁表
