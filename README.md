# 短视频平台创作者资源重配分析

## 问题与判断
在流量和现金激励受限的前提下，核心经营问题是：

> 平台当前的资源分配，是否持续流向更可能形成稳定供给、留存和收入的创作者？

当前判断如下：

- 资源损耗主要集中在**首发后的持续经营阶段**；
- 资源分配仍带有明显的**历史曝光惯性**，曝光集中度高于价值集中度；
- 通过“**准入 → 排序 → 复核**”的轻量校正，可以把一部分资源从低效率曝光中挪出，补向更高质量的稳定供给；
- 离线结果支持进入**分层灰度实验**，不支持直接跳过线上验证。

---

## 数据范围
- 创作者样本：**18,742**
- 内容样本：**326,418**
- 周度活跃记录：**299,872**
- 周度激励记录：**299,872**
- 样本性质：**合成样本**

---

## 核心发现
- Top 20% 创作者拿走 **42.9%** 曝光，但只贡献 **30.2%** 收入，资源集中度高于价值集中度。
- **高曝光低变现** 创作者共 **2,620** 人，占创作者 **14.0%**，却占曝光 **35.0%**，是主要资源沉没点。
- **高潜低激励** 创作者共 **1,297** 人，占创作者 **6.9%**，但 **30 日留存 62.4%**、**千次曝光收入 98.1**，存在识别滞后。
- 在 **Top 4,200** 同规模投放下，新规则使 **30 日留存率** 从 **48.7%** 提升到 **58.1%**，提升 **9.4 个百分点**；使 **单位激励收入** 从 **69.6** 提升到 **78.4**，提升 **12.7%**。
- **人均现金激励** 从 **550** 升到 **625**。当前结果更接近结构性资源重配，不能解读为单纯降本。
- Bootstrap 95% 区间显示：留存提升为 **8.8–10.1 个百分点**，单位激励收入提升为 **9.6%–16.0%**。
- 新旧 Top 4,200 名单重合率约 **72.2%**，属于轻量校正。

## 策略口径
资源分配按三步执行：

1. **准入过滤**：剔除明显不适合继续强扶持的人群；
2. **排序加权**：保留旧规则主体，同时补入留存、效率和低资源占用信号；
3. **名单复核**：对拟降配对象做灰度观察和复核。

首轮灰度采用的排序口径为：

```text
新优先级分数
= 0.60 × 原规则分位
+ 0.20 × 留存分位
+ 0.15 × 变现效率分位（缩尾后）
+ 0.05 × 低资源占用分位
```

采用该口径的依据是其同时满足三类约束：

- 名单重合率处于可执行区间；
- 30 日留存与单位激励收入同时改善；
- 不依赖极端削弱旧规则权重来换取表面提升。

---

## 候选口径取舍
`outputs/weight_sensitivity_screen.csv` 中仅保留首轮灰度最相关的候选组合。离线提升更高的组合存在；当前组合优先满足以下平衡：

- 名单替换幅度；
- 主指标改善幅度；
- 单人成本抬升速度；
- 运营复核和灰度承接能力。

## 动作建议
| 对象 | 动作 | 节奏 | 责任方 |
|---|---|---|---|
| 高潜低激励 | 增配曝光与激励，进入快速晋升池 | 每 7 天复评 | 策略 DA / 运营 |
| 高曝光低变现 | 进入降配阈值与复核队列 | 每 7 天复核 | 运营 / 风控 |
| 高价值稳定供给 | 保持稳定扶持 | 月度复盘 | 运营 |
| 一般供给 / 低价值供给 | 以内容诊断和成长任务为主 | 双周复盘 | 运营 |

---

## 解释边界
- 历史回测用于判断方向，不直接替代线上因果识别；
- 补入名单在历史上并未真实获得相同资源，因此离线提升更接近方向证据；
- 不同预算规模和内容垂类下方向基本一致，但收益释放幅度存在差异；
- 正式扩量前，仍需通过 **creator 级分层随机实验** 验证主结果与护栏表现。
