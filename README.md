# 创作者资源重配分析

这个项目讨论的是：在预算受限的前提下，平台是否把资源持续配置给了更可能形成稳定供给、留存和收入的创作者。

## 当前判断

- 供给损耗主要集中在首发后的持续经营阶段。
- Top 20% 创作者拿走 42.9% 曝光，只贡献 30.2% 收入，资源分配仍带有历史曝光惯性。
- 固定 Top 4,200 的历史回测下，新规则相对旧规则带来：
  - 30 日留存率：48.7% → 58.1%（+9.4ppt）
  - 单位激励收入：69.6 → 78.4（+12.7%）
  - 人均现金激励：550 → 625（+13.6%）

## 结论

离线诊断与同规模回测支持进入 creator 级分层灰度实验；是否扩量，由线上主指标与护栏决定。
