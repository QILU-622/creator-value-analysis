# 创作者资源重配分析报告

## 一、问题定义

在流量和现金激励受限的前提下，需要判断平台资源是否持续流向更可能形成稳定供给、留存和收入的创作者。当前离线证据支持进入首轮灰度，不支持直接跳过线上验证全量切换。

## 二、关键发现

1. 主要损耗发生在首发后的持续经营阶段。
2. Top 20% 创作者拿走 42.9% 曝光，只贡献 30.2% 收入，资源集中度高于价值集中度。
3. 高曝光低变现与高潜低激励是首轮动作最敏感的两类人群。
4. 固定 Top 4,200 回测下，新规则带来 30 日留存 +9.4ppt、单位激励收入 +12.7%、人均现金激励 +13.6%。
5. 新旧名单重合率约 72.2%，属于轻量校正。

## 三、首轮口径

```text
新优先级分数
= 0.60 × 原规则分位
+ 0.20 × 留存分位
+ 0.15 × 变现效率分位（缩尾后）
+ 0.05 × 低资源占用分位
```

首轮口径兼顾名单迁移速度、运营承接压力和主指标方向。

## 四、读数边界

- 历史回测只用于判断方向，不替代线上因果识别。
- 补入名单在历史上并未真实获得相同资源，因此离线 uplift 更接近方向证据。
- 推荐外溢、观察窗截断、新鲜感效应和跨池迁移污染，都需要通过正式实验控制。

## 五、上线建议

建议采用 creator 级分层随机实验；在主指标不为负、单位激励收入保持为正且关键护栏无明显恶化的前提下，按池和预算档位扩量。
