# 创作者资源重配分析：从离线诊断到实验设计

## 执行摘要

这个项目讨论的核心问题是：在预算受限的前提下，平台是否把资源持续配置给了更可能形成稳定供给、留存和收入的创作者。

当前离线证据支持进入首轮灰度，不支持跳过线上验证直接全量切换。

## 数据范围

| 数据 | 规模 |
|---|---:|
| 创作者样本 | 18,742 |
| 内容样本 | 326,418 |
| 周度活跃记录 | 299,872 |
| 周度激励记录 | 299,872 |
| 样本性质 | 合成样本 |

## 解释边界

- 旧规则与历史资源倾斜高度相关，因此带有路径依赖。
- 离线样本无法完全剥离运营干预、活动资源和季节波动。
- 补入名单在历史上没有真实拿到同等资源，因此不能把离线 uplift 直接写成真实增量。

## 规则口径

首轮灰度评分公式：

```text
新优先级分数
= 0.60 × 原规则分位
+ 0.20 × 留存分位
+ 0.15 × 变现效率分位（缩尾后）
+ 0.05 × 低资源占用分位
```

候选方案中，D 方案被用于首轮灰度。原因是它在名单迁移幅度、主指标改善和执行承接压力之间更平衡。

## Top 4,200 同规模回测

| 指标 | 旧规则 | 新规则 | 变化 |
|---|---:|---:|---:|
| 30 日留存率 | 48.7% | 58.1% | 9.4ppt |
| 单位激励收入 | 69.6 | 78.4 | 12.7% |
| 人均现金激励 | 550 | 625 | 13.6% |

## 扩量视角

随着 Top N 扩大，留存与单位激励收入改善幅度逐步下降，因此扩量更适合分阶段推进。

## 实验设计

- 随机化单位：creator 级分层随机实验
- 主指标：30 日留存率；单位激励收入
- 护栏：人均现金激励；供给流失率；投诉率；作弊率；内容安全
- 主读数原则：intent-to-treat 为主
- 扩量条件：主指标不为负、单位激励收入保持为正、关键护栏无明显恶化
- 回滚条件：SRM / 分流漂移显著、单位激励收入转负、投诉/作弊/安全恶化

## 有效性威胁

- 历史回测不是因果识别
- SRM / 分流漂移
- 推荐外溢
- 观察窗截断
- 新鲜感效应
- 跨池迁移污染

样本说明：数据为合成样本；离线结果仅用于判断是否进入实验，不替代线上因果识别。
